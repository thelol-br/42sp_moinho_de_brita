# Comandos usados:
```shell
truncate -s 40 testShell00
touch -t 202106012342 testShell00
chmod u=r,g=rx,o=rx testShell00
tar -cf testShell00.tar testShell00 --remove-files
```

## truncate
Uso o truncate para criar um arquivo vazio e a flag -s alterar o seu tamanho para 40 bytes.

## touch
Uso o touch com a flag -t para alterar a data de criação do arquivo.

[[CC]YY]MMDDhhmm[.ss]
CC: First two digit of the year
YY: Last two digits of the year
MM: Month (two-digit numeric month)
DD: Day (two-digit numeric day i.e. day of month)
hh: Hour
mm: Minutes
ss: Seconds

## chmod
Uso o chmod para alterar as permissões do arquivo.
u = user
g = group
o = other

r = read
w = write
x = execute

## tar
Uso o tar para armazenar o arquivo original mantendo as datas e permissões. As flags -c e -f, podem ser concatenadas em -cf, onde eu informo o ficheiro que será criado e qual o arquivo será armazenado nele em sua respectiva ordem. Por último, uso o --remove-files para remover os originais.
