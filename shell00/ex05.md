# Comandos usados:
```shell
#!/bin/sh
git ls-files -io --exclude-standard
```

## git ls-files
O git ls-files mostra informações sobre os arquivos no índice da worktree. A flag -io, serve para mostrar somente os arquivos **i**gnorados utilizando **o**utros arquivos e dando um pattern, que são os --exclude-standard, que usa o arquivo .gitignore em cada pasta, inclusive no diretório raiz do repositório.
