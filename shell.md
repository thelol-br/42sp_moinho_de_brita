ctrl + a    começo da linha.
ctrl + e    fim da linha.
ctrl + c    apagar a linha.
ctrl + l    limpar a tela.
pwd         onde você está no filesystem.
ls          listar todo o conteúdo **VISÍVEL** do diretório que estamos.
ls -p       flag p serve para ajudar a diferenciar arquivos de pastas.
ls -l       flag l serve para listar dará os detalhes de cada arquivo do diretório.
ls -la      flag la serve para listar todos os arquivos, inclusive os **OCULTOS**.
mkdir       criar um novo diretório.
rmdir       remove o diretório **VAZIO**.
cd          troca o diretório.
