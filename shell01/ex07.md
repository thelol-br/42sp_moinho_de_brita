# Comandos executados:
```shell
cat /etc/passwd | awk -F':' '{print $1}' | sed -n '0~2p' | rev | sort -r | sed -n ""$FT_LINE1","$FT_LINE2"p" | tr "\n" "," | sed 's/,/, /g' | sed 's/..$/./'
```

## Explicação dos comandos:
1. ```cat /etc/passwd``` Uso o cat para concatenar o arquivo passwd e printar no stdout.
2. ```awk -F':' '{print $1}'``` AWK é uma linguagem de programação interpretada que é, geralmente, usada para deixar os scripts de shell mais poderosos e com mais recursos. É usada mais para processar dados nos textos e operações com arquivos.<https://www.vivaolinux.com.br/artigo/AWK-Introducao>
Neste caso, utilizei a flag -F para indicar que o espaçador das colunas são dados como ":". Todo script de awl deve vir entre aspas simples ou duplas e chaves. O comando ```print $1``` serve para printar a primeira coluna somente.
3. ```sed -n '0~2p'``` sed é um editor de fluxo para filtrar e transformar textos. A flag -n faz com que o sed pare de imprimir linhas automaticamente, a não ser que os comandos `p` ou `l` sejam informados. O comando `'0~2p'` instrui o sed para imprimir uma linha a cada duas. Também poderia ser substituído por `'p;n'` com a mesma função.
4. ```rev``` O comando rev serve para inverter os caracteres de uma linha.
5. ```sort -r``` O comando sort serve para ordernar linhas de textos. Usando a flag -r, ele ordena inversamente, ou seja, [z-a].
6. ```sed -n ""$FT_LINE1","$FT_LINE2"p"``` Sed -n tem a mesma função anterior, porém, utiliza-se o limitador de colunas os valores das variáveis de ambiente FT_LINE1 e FT_LINE2. Para este comando funcionar corretamente em sed, deve-se escapar com duplas aspas duplas, pois estamos trabalhando em um sistema UNIX, caso contrário, poderia ter sido utilizado aspas simples.
7. ```tr "\n" ","``` O comando tr serve para manipular o texto de uma linha. Compõe-se com o primeiro membro o que eu quero mudar, que no caso é uma nova linha (`\n`) por vírgulas.
8. ```sed 's/,/, /g'``` Neste caso, utilizamos o sed com o comando `s/` para indicar uma substituição das vírgulas por `vírgula espaço` e o registrador `g` para indicar que é uma substituição global, ou seja, que deve ser feita em toda a linha.
9. ```sed 's/..$/./'``` Realizados uma substituição para trocar os dois últimos membros, sejam eles quais forem, de uma linha por um `.`. O `$` significa fim da linha e/ou começar pelo fim.
