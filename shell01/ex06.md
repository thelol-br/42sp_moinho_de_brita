# Comandos usados:
```shell
#!/bin/sh
ls -l | sed -n 'p;n'
```

## Explicação dos comandos:
1. ```ls -l``` = ls serve para listar o conteúdo de diretórios. A flag -l serve para utilizar uma listagem longa.
2. ```sed -n 'p;n'``` = sed é um editor de fluxo para filtrar e transformar textos. A flag -n foi utilizado para impedir que o sed imprima na saída, a não ser que tenha sido especificado pelos comandos p ou l. 
O comando "p" serve para informar o sed de voltar a imprimir cada linha apenas uma vez; O ";" serve como separador de comandos simples; O comando "n" informa ao sed para ir para a próxima linha.
