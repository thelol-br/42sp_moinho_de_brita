# Comandos utilizados:
```shell
#!/bin/sh
groups $FT_USER | tr " " "," | tr -d "\n" | cat -e
```

# Criação do teste:
```shell
export FT_USER=staff
```

## O que cada comando faz:
1. groups $FT_USER - O $ chama o valor da variável.
2. tr substitui cada espaço vazio por vírgula.
3. tr remove a quebra de linha.
4. cat -e faz o print conforme instruções do projeto.
