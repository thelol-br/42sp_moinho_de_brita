# Comandos usados:
```shell
echo -n 42 > '"\?$*'"'"'MaRViN'"'"'*$?\"'
```

## Explicação dos comandos:
1. echo serve para mostrar uma linha de texto, mas no meu caso, eu utilizei para criar um arquivo, adicionar 42 dentro e através da flag -n, não adicionar a quebra de linha que é padrão do echo.

2. Deveríamos criar um arquivo com um nome que utilizava caracteres especiais. Então eu usei a técnica de escape para utilizar os caracteres especiais como palavras e não como comandos do shell.
