# Comandos usados:
```shell
#!/bin/sh
ifconfig | grep -i ether | cut -c 15- | cut -c -17
```

## Explicação dos comandos:
1. ```ifconfig``` = Serve para configurar as interfaces de rede ou debugá-las, porém, se for usado sem nenhuma flag, ele lista todas as interfaces de rede ativa.
2. ```grep -i ether``` = grep procura por padrões dentro de um arquivo ou output. A flag -i é usada para que o padrão utilizado seja caso insensitivo, ou seja, procurará por ether, ETHER ou EtHeR.
3. ```cut -c 15-``` = cut remove partes selecionadas do arquivo. A flag -c serve para dizer que são SOMENTE estes CARACTERES e o 15- indica que deve-se remover 15 caracteres da esquerda para a direita ou do começo ao final.
3. ```cut -c -17``` = a diferença do comando atual para o explicado acima é que a ordem de ínicio de remoção dos caracteres dá-se após os 17 primeiros caracteres até o final, ou seja, mantem-se os 17 primeiros e removem-se os outros.
